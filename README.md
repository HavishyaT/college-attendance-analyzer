# College Attendance Analyzer

A Java Swing desktop application that helps analyze student attendance and alerts if attendance falls below 75%.

## Features

- Mark student attendance
- Visual attendance chart
- Attendance percentage calculation
- Alerts when attendance is low
- Predict if 75% attendance can still be achieved

## Technologies Used

- Java
- Swing (GUI)
- MVC Architecture

## Project Structure

Model
- Student
- Subject
- AttendanceLog

View
- AttendanceChartView
- AlertsView

Controller
- AttendanceController
